SCENARIO
You are an IT Support Admistrator Specialist and are charged with the task of creating email accounts for new hires.

Your application should do the following:
Generate an email with the following syntax: firstname.lastname@department.company.com
Determine the department (Sales, Development, Accounting) if none, leave blank
Generate a random string for a passwort
Have set methods to change passwort, set the mailbox capacity, and define an altenate email address
Have get methods to display the name, email, mailbox
